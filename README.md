# jxlgui
Windows Desktop App to convert images to JPEG XL files

The input can be PNG, APNG, GIF, JPEG, PPM, PFM, or PGX, a JPEG XL (JXL) file will be converted to PNG with ICC, JPG, or PPM/PFM.
